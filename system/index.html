Linux Commands Cheat Sheet

Quick reference for managing the game server directory at /srv/games/icarus

Navigation
cd /srv/games/icarus

Go to game server directory

ls -lah

List all files with details and sizes

pwd

Show current directory path

File Operations
cat filename.txt

View file contents

tail -f logfile.log

Watch log file in real-time (Ctrl+C to exit)

grep "error" logfile.log

Search for "error" in log file

Disk Usage
du -sh *

Show size of each folder/file

df -h

Show disk space usage

Process Management
ps aux | grep icarus

Find Icarus server processes

top

View system resources (q to quit)

Permissions
ls -l

View file permissions

chmod +x script.sh

Make script executable

Server Configuration
nano Icarus/Saved/Config/WindowsServer/ServerSettings.ini

Edit server settings (Ctrl+X to save and exit)

nano start.sh

Edit server startup script

About start.sh:

  • Launches Icarus server using Wine/Proton compatibility layer
  • Manages tmux session for background server operation
  • Checks for existing server processes to prevent duplicates
  • Logs startup information for troubleshooting
Caution: Be careful with commands that modify or delete files. Always double-check before running destructive operations.
